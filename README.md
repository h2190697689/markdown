# Markdown

## 标题(共6级)
```
# 一级标题
## 二级标题
### 三级标题
```

## 列表
```
1. 有序列表
- 圆形无序列表
* 圆环无序列表
```

## 引用
```
> 这里是引用类型
```

## 图片与链接
```
![]()  图片链接
[]()   文本链接
<https:www.baidu.com>   文件链接(没有说明文字)
```

## 加粗
```
** text **  
```

## 斜体
```
* text *  
```

## 斜体加粗
```
*** text ***  
```

## 删除线
```
~~ text ~~
```

## 下划线
```
<u>text</u>
```

## 分隔线
```
***
* * *
*****
- - -
----------
```

## 代码
```
    ` code `  单行代码
    
    ```
      lines code   多行代码
    ```
```



## 表格(:在哪边表示在哪边居中)
```
| HEAD1 | HEAD2 | HEAD3 |
| ----- |:-----:| -----:|
| BODY1 | BODY2 | BODY3 |
```

## 脚注
```
    脚注[^footer]
    [^footer]: you are not rookie, you are the preson who is running on the way
```
